## SFM REST API Specifications

This repository contains the OpenAPI specification for the SOTO Fleet Manager (SFM). In order to connect your Warehouse Management System (WMS) with the SFM,
you will need to call the endpoints specified in the `wms_to_SFM` specification. Additionally you will need to provide the endpoints in your system which are
specified in the `SFM_to_wms` specification to enable bidirectional communication.
